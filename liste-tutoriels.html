Voici les tutoriels de développement-web que je trouve cool

  • W3Schools - Un classique pour apprendre les bases du HTML, CSS et JavaScript.
  • MDN Web Docs - Une ressource complète pour les développeurs web de tous niveaux.
  • FreeCodeCamp - Un site interactif avec des projets pratiques pour apprendre le développement web.
  • Codecademy - Des cours interactifs pour apprendre à coder dans différents langages.
  • Udemy - Une plateforme de cours en ligne avec une vaste sélection de sujets.
  • Coursera - Des cours en ligne proposés par des universités et institutions renommées.
  • edX - Une autre plateforme de cours en ligne avec des programmes de qualité.
  • Khan Academy - Des leçons interactives sur divers sujets, y compris la programmation.
  • Udacity - Des programmes de nanodegrees axés sur les compétences pratiques.
  • SoloLearn - Une application mobile et un site web pour apprendre à coder en déplacement.
